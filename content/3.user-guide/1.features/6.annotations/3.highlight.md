### 📝 How to use Highlight

🧾 Click the Shapes Annotation icon to access the drawing tools.

🖍️ Right-click to enable the highlight to apply the highlight color to the selected content.

⚡ Select the text you highlithed again to switch to a different color or remove the highlight entirely.

💬 After highlighting, Start a Thread to create a conversation tied directly to the highlighted section.

### 💡 Tips 

* Use highlights to guide attention, organize long messages, or visually separate key concepts.

* Highlighting before starting a thread helps others understand exactly what part you’re referring to.

* Keep highlighting minimal so the Dot doesn’t become visually overwhelming.
