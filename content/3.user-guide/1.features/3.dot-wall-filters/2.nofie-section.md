A built-in search tool that allows users to quickly find specific messages using keywords, improving navigation and information retrieval.

### 💡 Tips

* Use Pinned to save messages you'll need later. It keeps your workspace clean and focused.

* Try the Search Bar for quick filtering instead of scrolling through long threads.

* Switch to Nofie when you only want to see messages that require your attention.
