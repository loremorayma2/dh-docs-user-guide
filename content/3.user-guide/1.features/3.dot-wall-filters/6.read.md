### Step-by-Step

🧾 Click the Read button to filter messages that have already been read.

📄 Review the messages to revisit important updates or conversations.

🔍 Use the search bar to quickly find specific read messages by keywords.

🧾 If needed, you can mark a message as unread to revisit it later or trigger notifications.

### 💡Tips

* Use the Read section to review past activity and ensure nothing important is overlooked.

* Combine with Pinned Dots to reference frequently needed messages quickly.

* The search feature in the Read section can save time when looking for older messages.
