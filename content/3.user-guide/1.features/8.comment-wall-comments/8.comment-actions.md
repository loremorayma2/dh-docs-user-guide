### ✏️ Edit Comment

Edit a comment when you need to fix a typo, update information, or clarify your message.

#### 📝 How to Edit a Comment

🧾 Open the dot where the message you want to edit is located.

⚙️ Move your cursor over the message you want to edit.

✏️ Click the pencil icon to edit. Your comment becomes editable.

⌨️ Make your corrections or add new information.

💾 Press Enter or click Save to confirm.

### 🗑️ Delete Comment

Delete a comment when the message is no longer relevant or was added by mistake.

#### 📝 How to Delete a Comment

🧾 Open the dot where the message you want to delete is located.

⚙️ Click the three-dot menu.

🗑️ A confirmation message appears.

✔️ The comment is removed permanently.

### ↩️ Reply to a Comment

Reply to a comment to keep conversations organized and grouped by topic.

#### 📝 How to Reply

📍 Select the message you want to respond to.

↩️ The reply field opens directly under the parent comment.

⌨️ Add context or answer the comment.

💬 Press Enter to submit.

### ⭐ Set as Important

Mark a comment as important to highlight crucial information. Important comments stay visually marked so they stand out in long conversations.

#### 📝 How to Mark a Comment as Important

🧾 Find the message you want to highlight.

⚙️ Click the three-dot menu.

⭐ Select the option: “Set as Important”

🔄 You can remove the highlight anytime by selecting the option again.

### 💡 Tips

* Use edits for corrections instead of adding multiple follow-up messages.

* Delete only when necessary to keep conversation history clear.

* Use replies to keep threads organized and easy to follow.

* Perfect for key decisions, next steps, or critical information.
