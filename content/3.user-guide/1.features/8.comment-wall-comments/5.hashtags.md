### 📝 How to use Hashtags

🧾Navigate to the dot where you want to add a hashtag.

⌨️ In the comment box or annotation bar, type # to activate the hashtag field.

🏷️ Write a new hashtag name to create it instantly, or Select an existing hashtag from the dropdown suggestions.

📌 Once entered, the hashtag is automatically added to the dot and becomes searchable.

🌀 Any dots using the same hashtag will appear grouped in the Hashtag view or search filter.

🔎 Use the Hashtag Cloud to quickly find all dots connected to the same tag.

✏️ You can add multiple hashtags to a dot to create richer filtering and categorization.

### 💡 Tips

* Use hashtags to track themes like #BugReport, #UIReview, #Sprint12, or #Priority.

* Keep hashtags consistent across the team to maintain clean organization.

* Short, clear hashtags work best — avoid long phrases.
