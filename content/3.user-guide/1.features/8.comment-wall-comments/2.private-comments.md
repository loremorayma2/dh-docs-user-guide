### 📝 How to use Private Comments

🧾 Navigate to the dot where you want to start or view a private conversation.

👤 Open the Comments panel and choose the person you want to message privately.

🔄 Before sending the message, click the Private Comment icon to switch from public comments to a private comment.

🔒 Type your message and press Enter. Only you and the selected person will be able to see this conversation.

💬 Exchange replies back and forth just like a normal comment thread — but fully private.

### 💡 Tips

* Use private comments to share confidential feedback or context meant for one person.

* Ideal for reviewing drafts, giving early opinions, or clarifying misunderstandings privately.

* You can maintain multiple private threads with different users inside the same dot.
