This centralized space helps you stay organized, track updates, and quickly handle priority items. Whether you’re catching up on unread dots or managing ongoing tasks, the Dot Wall makes notification management simple and efficient.

### 📝 How to Use Dot Wall Notifications

##### 1. 🔍 Dot Wall Search

* Click the **Search** icon in the Dot Wall.
* Type a **keyword, phrase,** or **hashtag** to filter dots.
* The Dot Wall will display only dots matching your search.
* Use hashtags to find related dots more efficiently.

##### 2. 📘 Mark as Read

* Click the **three-dot menu** on a wall notification.
* Select **Mark as Read.**
* The dot will be marked as already reviewed.
* To mark several dots at once, you can select multiple notifications and apply the same action.

##### 3. 📩 Mark as Unread

* Click the **three-dot menu** on a wall notification.
* Select **Mark as Unread.**
* The dot will be saved as pending so you can review it later.
* Useful for priority dots or reminders, and can be applied to multiple notifications at the same time.

##### 4. 📌 Pin Dots

* Open the dot you want to pin from the Dot Wall.
* Click the **three-dot menu** inside the dot.
* Select **Pin.**
* The dot will move to the **Pinned Dots** section for quick access.
* To unpin, repeat the steps and select **Unpin Dot.**

##### 5. 🗑️ Delete Notification

* Click the **three-dot menu** on the notification you want to remove.
* Select **Delete Notification.**
* The dot will immediately disappear from your Wall.
* This deletes the actual dot.

### 💡Tips

* Use **Mark as Unread** to create quick reminders for dots you want to revisit.
* Pin high-priority dots to avoid losing them in a long list.
* Search using hashtags for better organization and topic tracking.
* Combine **Pin** + **Unread** for maximum visibility of urgent items.
