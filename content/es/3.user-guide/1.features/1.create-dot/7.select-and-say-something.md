### 🔵 Clic y Selecciona

Después de delinear la sección, puedes convertirla en un Dot para iniciar un hilo y compartirlo con otros.

Esto facilita marcar áreas específicas en la página y comunicarte de manera clara.

También puedes usar ALT + A para hacerlo de manera más rápida.
