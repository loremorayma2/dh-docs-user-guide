### 🟢 1. Desplázate hasta el final de la página

Puedes crear un Dot tomando una captura rápida directamente desde tu espacio de trabajo.

Se abrirá la barra de captura rápida. Mueve tu cursor hasta el final de la página en cualquier momento para capturar.

También puedes usar ALT + S para hacerlo de manera más rápida.

### 🟠 3. Clic y arrastrar

Captura contenido dentro de una ventana o componente seleccionado. Ideal para enfocarse en una herramienta o área específica de la aplicación. También puedes usar ALT + C.

  <div class="border rounded-lg shadow-md mt-4 h-60">

### 💡 Consejos

* Cada captura se convierte automáticamente en un Dot, listo para anotar, organizar y compartir con tu equipo.

* Usa Alt + S para capturas de página completa; es la forma más rápida de capturar todo lo que se ve.

* Prueba Alt + A cuando solo necesites mostrar un elemento específico o un error puntual.

* Después de tomar una captura, agrega una breve descripción antes de que olvides el contexto.
