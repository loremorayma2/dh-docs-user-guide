### 🔵 Clic y Resalta

Resalta una parte específica de tu pantalla.

Es ideal para capturar un detalle o sección concreta. También puedes usar la herramienta de lápiz para resaltar o marcar cualquier área que quieras enfatizar en la página.

También puedes usar ALT + A para hacerlo de manera más rápida.
