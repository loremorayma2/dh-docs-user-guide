💡 Consejos

* Usa stickers para mostrar reacciones o reconocer comentarios rápidamente — por ejemplo: ✅ Aprobado, ⚠️ Requiere atención, 🌟 ¡Gran idea!

* Combina stickers con anotaciones o emojis para reforzar tu mensaje visualmente.

* Mantén los stickers al mínimo para evitar sobrecargar tu dot — generalmente 1–3 por captura es suficiente.

* Usa stickers de manera consistente en todo tu equipo para que el feedback sea fácilmente reconocible.
