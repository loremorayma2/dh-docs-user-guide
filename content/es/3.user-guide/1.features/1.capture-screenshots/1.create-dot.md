### 📸 Capturar Pantallas

Puedes crear un Dot tomando una captura de pantalla directamente desde tu espacio de trabajo.  
Hay tres formas sencillas de hacerlo: elige la que mejor se adapte a tu flujo de trabajo.

---

### 🟢 1. Pasa el cursor al final de la página

Esto abrirá la barra de captura rápida. Lleva tu cursor hasta el final de la página en cualquier momento para capturar.  
También puedes usar **ALT + S**.

### 🔵 2. Haz clic y arrastra

Selecciona una parte específica de tu pantalla.  
Ideal para capturar un detalle o una sección; puedes arrastrar con la herramienta de lápiz para marcar un área específica de la página.  
También puedes usar **ALT + A**.

<template>
  <div class="border rounded-lg shadow-md mt-4 h-60">
    <iframe src="/quickstart-capture.png" class="rounded-lg w-full h-full" scrolling="no" frameborder="0"></iframe>
  </div>
</template>

---

### 🟠 3. Captura una ventana o componente

Captura el contenido dentro de una ventana o componente específico.  
Perfecto para enfocarte en una herramienta o área concreta de la aplicación.  
También puedes usar **ALT + C**.

  <div class="border rounded-lg shadow-md mt-4 h-60">

---

### 💡 Consejos

* Cada captura se convierte automáticamente en un **Dot**, listo para ser anotado, organizado y compartido con tu equipo.  
* Usa **Alt + S** para capturas de página completa: es la forma más rápida de obtener todo lo que ves.  
* Prueba **Alt + A** cuando solo necesites mostrar un elemento o error específico.  
* Después de tomar una captura, añade una breve descripción para no olvidar el contexto.
