💡 Consejos

* Usa emojis para reaccionar a los comentarios — por ejemplo: 👍 para acuerdo, ❤️ para apreciación, 😂 para algo divertido.

* Combina emojis con anotaciones para llamar la atención sobre detalles importantes.

* No los uses en exceso — elige 1–2 por Dot para mantener el mensaje claro.

* Intenta usar colores o estilos de emoji consistentes en tu equipo para facilitar el reconocimiento.
