💡 Consejos

* Usa stickers para mostrar reacciones o reconocer comentarios rápidamente — por ejemplo: ✅ Aprobado, ⚠️ Necesita atención, 🌟 ¡Gran idea!

* Combina stickers con anotaciones o emojis para reforzar tu mensaje visualmente.

* Mantén los stickers al mínimo para evitar sobrecargar tu Dot — 1–3 por captura suelen ser suficientes.

* Usa stickers de manera consistente en tu equipo para que los comentarios sean fácilmente reconocibles.
