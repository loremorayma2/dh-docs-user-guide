    # Preguntas Frecuentes <strong class="text-pink-600">(FAQ)</strong> ❓
  ::
  ::div{class="flex flex-col justify-start items-start w-full"}
    ::accordion{default-value="first-item" collapsible class="flex w-full flex-col"}

      ::accordion-item{value="first-item"}
      #title
      ###### 1. ¿Qué es DOT APP?

      #content
      **DOT APP** es una plataforma de colaboración visual que te permite capturar pantallas, anotar y compartir ideas en tiempo real, desde tu escritorio o navegador.
      ::

      ::accordion-item
      #title
      ###### 2. ¿Con qué sistemas operativos es compatible?

      #content
      - **Windows 10 o superior**  
      - **macOS 11 o superior**  
      - Extensiones disponibles para **Chrome** y **Edge**
      ::

      ::accordion-item
      #title
      ###### 3. ¿Puedo trabajar sin conexión a Internet?

      #content
      Sí, puedes capturar y anotar imágenes sin conexión, pero la **colaboración en tiempo real** requiere una conexión a Internet.
      ::

      ::accordion-item
      #title
      ###### 4. ¿Cómo comparto mis capturas con el equipo?

      #content
      Puedes usar el **enlace de colaboración** desde el panel principal o sincronizar automáticamente con tu equipo si todos tienen DOT APP instalado.
      ::

      ::accordion-item
      #title
      ###### 5. ¿Cómo puedo crear un nuevo hilo?

      #content
      Toma una captura usando la opción **wall dot** en el panel principal, escribe tu mensaje o comentario, y selecciona los miembros que participarán.  
      Una vez creado, todos los participantes podrán contribuir con ideas y comentarios de inmediato.
      ::

      ::accordion-item
      #title
      ###### 6. ¿Puedo trabajar en varios hilos al mismo tiempo?

      #content
      Sí. Puedes tener múltiples hilos activos y cambiar entre ellos desde el panel lateral.  
      Cada hilo mantiene un historial independiente de mensajes y actividades.
      ::

      ::accordion-item
      #title
      ###### 7. ¿Cómo sé quién está conectado y colaborando en un hilo?

      #content
      En la parte superior del **wall dot** verás una lista de usuarios conectados en tiempo real.  
      También observarás un indicador de actividad cuando alguien esté escribiendo o realizando cambios.
      ::

      ::accordion-item
      #title
      ###### 8. ¿Los cambios que hacemos se guardan automáticamente?

      #content
      Sí. Toda la información compartida en los hilos se guarda en tiempo real y de forma automática para evitar pérdida de datos.
      ::

      ::accordion-item
      #title
      ###### 9. ¿Puedo mencionar a mis compañeros dentro de un hilo?

      #content
      Sí. Usa el símbolo **@** seguido del nombre de usuario para notificarlos directamente dentro del hilo.
      ::

      ::accordion-item
      #title
      ###### 10. ¿Puedo adjuntar archivos o documentos a un hilo?

      #content
      Sí. Puedes arrastrar y soltar archivos dentro del hilo o usar el botón **“Adjuntar”**.  
      Todos los miembros con acceso podrán verlos y descargarlos.
      ::

      ::accordion-item
      #title
      ###### 11. ¿Quién puede ver los hilos que creo?

      #content
      Por defecto, solo los usuarios que invites tendrán acceso.
      ::

      ::accordion-item
      #title
      ###### 12. No veo las actualizaciones en tiempo real. ¿Qué debo hacer?

      #content
      Verifica tu conexión a Internet.  
      Si el problema persiste, actualiza la página o cierra sesión y vuelve a ingresar.
      ::

      ::accordion-item
      #title
      ###### 13. Olvidé mi contraseña. ¿Cómo la recupero?

      #content
      Haz clic en **“¿Olvidaste tu contraseña?”** en la pantalla de inicio de sesión y sigue los pasos para restablecerla usando tu correo electrónico registrado.
      ::
